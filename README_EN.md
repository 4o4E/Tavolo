# Tavolo

English | [简体中文](README.md)

`Tavolo` is pronounced `TAH-vo-lo`. The name comes from Esperanto and means "layer".

## Project Goal

Tavolo is built for image processing and offline rendering in server environments without display output.
It does not depend on desktop windows or interactive canvases. Instead, it renders input data, fonts, image assets, and drawing DSL content into images.
The core rendering capability is powered by [Skiko](https://github.com/JetBrains/skiko).

Tavolo is suitable for:

- Generating static images, cards, charts, and SVG mixed layouts on a headless server.
- Processing images or GIFs frame by frame offline to generate memes, filters, and animated effects.
- Exposing image processing commands over HTTP for bots or business systems.
- Parsing BDF bitmap fonts for dot-matrix text and pixel-style image generation.

## Modules

| Module | Purpose |
| --- | --- |
| [`graphics`](graphics/src/main/kotlin) | Compose-style drawing DSL with layout, text, images, SVG, charts, Modifier effects, and 3D rendering. |
| [`gif-codec`](gif-codec/src/main/kotlin) | GIF encoding/decoding and frame processing. Part of the implementation references [`cssxsh/mirai-skia-plugin`](https://github.com/cssxsh/mirai-skia-plugin). |
| [`core`](core/src/main/kotlin) | Image processing commands, meme generators, and input-driven image generation. |
| [`bdf-parser`](bdf-parser/src/main/kotlin) | BDF bitmap font parser. |
| [`http-server`](http-server) | HTTP command service for command discovery and image execution APIs. |
| [`http-client`](http-client) | Client for the HTTP command service. |

## Dependency

Check [Releases](https://github.com/4o4E/Tavolo/releases) for available versions. Release builds are published to GitHub Packages.

GitHub Packages requires authentication when resolving Maven packages. Configure your user-level `~/.gradle/gradle.properties` first:

```properties
gpr.user=your GitHub username
gpr.key=a GitHub classic token with read:packages permission
```

Then add the repository and dependencies:

```kotlin
val version = "2.0.1"

repositories {
    mavenCentral()
    maven {
        name = "GitHubPackages"
        url = uri("https://maven.pkg.github.com/4o4E/Tavolo")
        credentials {
            username = providers.gradleProperty("gpr.user")
                .orElse(providers.environmentVariable("GITHUB_ACTOR"))
                .orNull
            password = providers.gradleProperty("gpr.key")
                .orElse(providers.environmentVariable("GITHUB_TOKEN"))
                .orNull
        }
    }
}

dependencies {
    implementation("top.e404.tavolo:tavolo-core:${version}")
    implementation("top.e404.tavolo:tavolo-graphics:${version}")
    implementation("top.e404.tavolo:tavolo-gif-codec:${version}")
    implementation("top.e404.tavolo:tavolo-common:${version}")
}
```

## Local Verification

Make sure Java 11 or later is installed and configured first.

Run the regular test suite:

```shell
./gradlew test
```

Generate only the README Hello World example image:

```shell
./gradlew :graphics:manualTest --tests "*ComposeHelloWorldManualTest"
```

Manual test outputs are written to `run/out`.
