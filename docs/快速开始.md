# 快速开始

本文面向只想先生成一张图片的调用方，更多布局规则和组件细节见 [graphics README](../graphics/README.md)。

## 环境要求

- JDK 11 或更高版本。
- Gradle Kotlin DSL 项目，或其他可以引入 Maven 依赖的 JVM 项目。
- 如果使用 snapshot 版本，需要配置 Sonatype snapshot 仓库。

## 引入依赖

```kotlin
val tavoloVersion = "2.1.0"

repositories {
    mavenCentral()
    maven("https://maven.pkg.jetbrains.space/public/p/compose/dev")
    // 仅在使用 -SNAPSHOT 版本时需要。
    maven("https://central.sonatype.com/repository/maven-snapshots/")
}

dependencies {
    implementation("top.e404.tavolo:tavolo-graphics:$tavoloVersion")
}
```

如果还需要图片处理、GIF 编码或 HTTP 指令服务，再按需加入：

```kotlin
dependencies {
    implementation("top.e404.tavolo:tavolo-core:$tavoloVersion")
    implementation("top.e404.tavolo:tavolo-gif-codec:$tavoloVersion")
    implementation("top.e404.tavolo:tavolo-http-server:$tavoloVersion")
}
```

## 生成第一张图片

```kotlin
import org.jetbrains.skia.Color
import org.jetbrains.skia.EncodedImageFormat
import org.jetbrains.skia.Image
import top.e404.tavolo.draw.compose.Modifier
import top.e404.tavolo.draw.compose.Shape
import top.e404.tavolo.draw.compose.TextModifier
import top.e404.tavolo.draw.compose.background
import top.e404.tavolo.draw.compose.box
import top.e404.tavolo.draw.compose.clip
import top.e404.tavolo.draw.compose.column
import top.e404.tavolo.draw.compose.font
import top.e404.tavolo.draw.compose.padding
import top.e404.tavolo.draw.compose.render
import top.e404.tavolo.draw.compose.size
import top.e404.tavolo.draw.compose.text
import top.e404.tavolo.util.FontManager
import java.io.File

fun main() {
    val uiFont = FontManager.registerSystem("ui", "Microsoft YaHei")
    val image = render(Color.TRANSPARENT) {
        column(
            modifier = Modifier
                .padding(24f)
                .clip(Shape.RoundedRect(24f))
                .background(Color.makeRGB(245, 247, 250))
                .padding(28f)
        ) {
            text(
                "Tavolo",
                textModifier = TextModifier
                    .font(fontSize = 42f, textColor = Color.makeRGB(32, 38, 46), fontFamily = uiFont)
            )
            text(
                "用 Compose 语法生成图片",
                modifier = Modifier.padding(top = 8f),
                textModifier = TextModifier
                    .font(fontSize = 22f, textColor = Color.makeRGB(91, 103, 120), fontFamily = uiFont)
            )
            box(
                modifier = Modifier
                    .padding(top = 22f)
                    .size(260f, 72f)
                    .clip(Shape.RoundedRect(18f))
                    .background(Color.makeRGB(48, 104, 255))
            ) {
                text(
                    "Hello Compose",
                    modifier = Modifier.padding(horizontal = 28f, vertical = 18f),
                    textModifier = TextModifier
                        .font(fontSize = 26f, textColor = Color.WHITE, fontFamily = uiFont)
                )
            }
        }
    }

    image.savePng(File("out/hello.png"))
}

private fun Image.savePng(file: File) {
    file.parentFile.mkdirs()
    val data = encodeToData(EncodedImageFormat.PNG) ?: error("PNG 编码失败")
    try {
        file.writeBytes(data.bytes)
    } finally {
        data.close()
        close()
    }
}
```

运行后会得到 `out/hello.png`。如果部署环境没有 `Microsoft YaHei`，可以换成已经安装的系统字体，或使用 `FontManager.registerFile("ui", File("font/NotoSansCJKsc-Regular.otf"))` 注册随程序分发的字体文件。

## 渲染 SVG

`svg` 组件会通过 Skia SVGDOM 渲染标准 SVG 内容。推荐提供 `viewBox`，这样不传 `Modifier.size` 时也能推导自然尺寸。

```kotlin
import org.jetbrains.skia.Color
import top.e404.tavolo.draw.compose.Modifier
import top.e404.tavolo.draw.compose.padding
import top.e404.tavolo.draw.compose.render
import top.e404.tavolo.draw.compose.sizeIn
import top.e404.tavolo.draw.compose.svg

val image = render(Color.TRANSPARENT) {
    svg(
        """
        <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 240 160">
            <rect x="8" y="8" width="224" height="144" rx="28" fill="#3068ff"/>
            <circle cx="78" cy="80" r="42" fill="#2bc37a"/>
            <path d="M120 120 L172 38 L218 120 Z" fill="#f5b130"/>
        </svg>
        """,
        Modifier
            .padding(20f)
            .sizeIn(maxWidth = 320f, maxHeight = 220f)
    )
}
```

如果 SVG 来自文件或网络响应，可以直接传 `ByteArray`：

```kotlin
svg(svgFile.readBytes(), Modifier.sizeIn(maxWidth = 320f, maxHeight = 220f))
```

## 运行项目示例

```shell
./gradlew :graphics:manualTest
```

```shell
./gradlew :graphics:manualTest --tests "*ComposeSvgManualTest"
```

## 排查布局

当图片尺寸或裁剪结果不符合预期时，可以用 `debugBaseElement` 输出测量后的树结构：

```kotlin
val root = Column()
render(Color.TRANSPARENT, root) {
    text("需要排查的内容")
}

val debug = StringBuilder()
debugBaseElement(0, root, debug)
println(debug.toString())
```

常见规则：

- `padding` 同时承担外部留白和内部留白，位置由链式顺序决定。
- `clip` 应放在 `background` 之前，背景才会被裁剪到目标形状内。
- `sizeIn` 适合限制最大宽高，SVG 和图片类组件会尽量保持比例。
